# Инструмент стеганографии

**LSB стеганография изображений с шифрованием AES-256**  
Прятать зашифрованные текстовые сообщения внутри PNG изображений. Безопасно, без потерь, незаметно глазу.

## Возможности
- **LSB Стеганография**: Скрытие данных в младших битах RGB пикселей
- **AES-256-CBC**: Военный уровень шифрования с PBKDF2
- **Поддержка PNG**: Формат без потерь сохраняет данные
- **Консольный интерфейс**: Простой CLI с проверкой ввода
- **Кроссплатформенность**: .NET 6+ (Windows/Linux/macOS)

## Установка

```bash
# Создать консольный проект
dotnet new console -n SteganographyTool
cd SteganographyTool

# Заменить Program.cs на код
# Или собрать готовый: dotnet publish -c Release -r win-x64 --self-contained
```

**Требования:**
- .NET 6.0+ SDK
- Windows (System.Drawing.Common) или Linux/macOS (libgdiplus)


## Как работает

```
Текст → AES-256 Шифрование → Base64 → LSB Встраивание → PNG
PNG → LSB Извлечение → Base64 Декодирование → AES Расшифровка → Текст
```

1. **Шифрование**: AES-256-CBC с рандомными salt/IV, PBKDF2(1000 итераций)
2. **Встраивание**: LSB каналов RGB (3 бита/пиксель). Изменение: ±1 интенсивность
3. **Извлечение**: Читает LSB до нулевого терминатора, авто-остановка

## Безопасность
```
Шифрование: AES-256-CBC (FIPS-140)
Вывод ключа: PBKDF2-SHA1 (1000 итераций)
Salt/IV: 32 байта рандом каждый раз
Устойчивость к стегоанализу: Базовый LSB (статистические атаки обнаруживают)
```

## Ограничения
- **Ёмкость**: ~0.375 символа на пиксель (только текст)
- **Формат**: Только PNG (требуется без потерь)
- **Обнаружение**: Статистический анализ выявляет LSB паттерны
- **Размер**: Большие изображения для длинных сообщений

## История разработки (7 коммитов)
```
1. Init: Консольное меню
2. LSB встраивание (EmbedText)
3. LSB извлечение (ExtractText)  
4. Исправление битов
5. XOR шифрование
6. Обновление до AES-256
7. UI доработка + v1.0 ✓
```
